AttendX is a modern, responsive **Attendance Management Dashboard** built using **React + Vite + Tailwind CSS**.  
It provides a clean interface to track, visualize, and manage attendance efficiently.

---

## Features

-  Interactive attendance dashboard
-  Beautiful charts using Recharts
-  Responsive UI with Tailwind CSS
-  Fast development with Vite
-  Clean and modular component structure

---

##  Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS
- **Charts:** Recharts
- **Bundler:** Vite
- **Language:** JavaScript

---

## Prerequisites

- Node.js 16 or later (recommended)  
- npm (or yarn / pnpm)  
- Optional: Docker for containerized builds
